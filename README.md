# Java Interface Payment System

## About this Project

I created this mini project while learning Java Interfaces.

The main goal of this project is to understand how interfaces work in Java and how multiple classes implement the same interface.

## Concepts Practiced

- Interface
- Multiple Interfaces
- Method Overriding
- Runtime Polymorphism

## Classes

- GooglePay
- PhonePe
- CreditCard
- Cash

## Interfaces

- Payment
- Receipt

## What I Learned

- Creating interfaces
- Implementing interfaces
- Runtime polymorphism
- Method overriding
- Multiple interface implementation
